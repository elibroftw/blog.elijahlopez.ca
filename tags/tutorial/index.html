For most errors like 403: Forbidden, try upgrading yt-dlp.

pip install --upgrade yt-dlp

  • Windows download

    • winget install yt-dlp
    • choco install yt-dlp
    • scoop install yt-dlp

    • brew install yt-dlp

  • Linux download
